# NoCheatItems - 高階作弊物品掃描與隔離系統

這是一個專為 Folia 架構設計的 Minecraft 插件，旨在精準偵測、掃描並沒收伺服器中的違規作弊物品（如：32K 武器、屬性誇張的裝備等）。

## 🌟 主要特色

* **Folia 完全相容**
* **動態黑名單管理**：直接透過遊戲內 GUI 介面管理要沒收的物品範本，隨放即偵測，拿走即撤銷。
* **高階特徵比對**：不僅比對名稱與附魔，還能偵測 Attribute Modifiers (屬性修改器)、Unbreakable (無法破壞) 等 1.21+ 組件資料。
* **隔離區系統 (Box)**：違規物品不再直接刪除，而是移至「隔離區」，防止管理員誤設名單導致誤刪，提供後悔藥機制。
* **管理員豁免**：持有管理權限者不受掃描影響，方便安全地設定範本。
* **詳細日誌記錄**：所有偵測與沒收行為都會非同步紀錄至 logs.txt。

## 🎮 指令說明

* `/nocheatitem blacklist` - 開啟黑名單管理庫，放入其中的物品將成為偵測範本。
* `/nocheatitem run` - 對全服在線玩家執行一次性沒收掃描，並移至隔離區。
* `/nocheatitem scan` - 對全服在線玩家執行一次性查看掃描（不沒收）。
* `/nocheatitem autorun` - 切換自動偵測開關。開啟後，玩家登入時會自動執行沒收掃描。
* `/nocheatitem box` - 開啟隔離區，管理員可在此查看或歸還被沒收的物品。
